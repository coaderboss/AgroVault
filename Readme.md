**GallaVault** is a premium, full-stack ERP and Point of Sale (POS) solution designed specifically for retail, agribusinesses, and wholesalers. It modernizes traditional ledger keeping (*Bahi Khata*) by combining hyper-fast billing, dynamic inventory management, and secure multi-tier user access into a sleek, mobile-first Progressive Web App (PWA).

---

## ✨ System Architecture & Core Capabilities

### 🛒 Advanced Point of Sale (POS)
* **Lightning-Fast Billing:** Dynamic unit conversions (Bags, Packets, Dozens to Base KG/Ltr) handled seamlessly on the backend.
* **WhatsApp Integration:** Generates and shares professional, beautifully formatted digital receipts directly to customers via WhatsApp natively.
* **Smart Cart Engine:** Real-time calculation of pricing, dues, and inventory allocation without page reloads.

### 📒 Ledger & Financial Tracking (Udhaar Management)
* **Micro-Transaction Tracking:** Keep track of full, partial, and pending payments effortlessly.
* **1-Click Bulk Pay:** Complex algorithm to allow customers to clear multiple pending invoices with a single lump-sum payment.
* **Automated Reminders:** Direct integration to send polite, formatted payment reminders to clients via WhatsApp.

### ⚡ Premium UI/UX & Data Entry
* **Excel-Style Bulk Onboarding:** A custom-built, keyboard-friendly data grid allowing business owners to onboard 10+ customers or suppliers in seconds using just the `Tab` key (bypassing tedious single-form entries).
* **Neo-Glassmorphism UI:** Built with Tailwind CSS and Lucide React icons for a highly polished, enterprise-grade feel.
* **1-Click Local Backup:** Client-side CSV generation allowing owners to download their entire customer ledger instantly for offline peace of mind.

### 🛡️ Enterprise-Grade Security & Authentication
* **Role-Based Access Control (RBAC):** Strict "OWNER" and "EMPLOYEE" tier system. Employees require a secure cryptographic `Shop Key` generated by the Owner to register and access the workspace.
* **Bulletproof Backend:** * Mitigated against **LPDos (Long Password Denial of Service)** via strict input length limits before bcrypt hashing.
  * Secured against **SQL/NoSQL Injection** using Prisma's parameterized queries.
  * Stateless, high-speed authentication using **JWT (JSON Web Tokens)**.

## 🛠️ Technology Stack Overview

**Frontend Architecture:**
* **Framework:** Next.js 14 (App Router)
* **Styling & Animation:** Tailwind CSS, Tailwind-Animate
* **Icons:** Lucide React
* **State Management:** React Hooks & Custom Contexts
* **PWA Ready:** Configured with Service Workers and Web App Manifest for native-like mobile installation.

**Backend Architecture:**
* **Runtime:** Node.js
* **Framework:** Express.js (RESTful API architecture)
* **Database:** PostgreSQL (Cloud-hosted via Neon.tech)
* **ORM:** Prisma Client
* **Security:** Bcrypt.js (Password hashing), JWT (Authorization)
